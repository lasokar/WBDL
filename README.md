## What is WBDL?
The **Web Browser Demon List (WBDL)** is a ranking of the hardest levels playable on the web-based version of Geometry Dash. The goal is simply to have fun, you shouldn't see this as an alternative to buying the actual game! (we do not condone piracy)

Levels are ranked based on their estimated difficulty. Players earn **List Points** by beating ranked levels, with harder levels awarding more points. This ranking system is inspired by other lists such as pointercrate.

WBDL uses a modded version of the geometrydash.com demo known as Web Dashers. It is not affiliated with RobTop Games. The project adds features such as custom level loading, extra objects, quality-of-life improvements, and tools for creating and playing custom levels.
